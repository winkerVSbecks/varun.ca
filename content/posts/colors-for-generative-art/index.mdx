## Use a colour scale

Chroma JS

```js
const colourScale = chroma
  .cubehelix()
  .rotations(3)
  .gamma(0.5)
  .scale()
  .correctLightness()
  .domain([-1, 1]);

const n = Random.noise3D(x, y, time);
context.fillStyle = colourScale(n);
```
